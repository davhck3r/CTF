**DOCKERLABS CHOCOLATELOVERS  **- https://dockerlabs.es/ 
##Dificultad: Fácil

###Instalación
Se obtuvo un archivo .zip desde la página oficial de Dockerlabs https://mega.nz/file/FbcgFS4J#_hcnLHn7jiXA-xlNkeXMRd92bJSaubMDeLIXZ-cMBrs

Una vez con el archivo, utilizamos la herramienta unzip para descomprimir y desplegamos el contenedor mediante el comando: 

```
sudo bash autodeploy.sh chocolatelovers.tar
```
